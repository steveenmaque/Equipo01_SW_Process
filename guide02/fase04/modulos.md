## Módulos

### Nueva Cotización

Este módulo permite registrar una nueva cotización para un cliente con sus respectivos datos, incluyendo datos como productos o inflables requeridos, cantidades, precios y observaciones. Genera un documento que puede visualizarse o imprimirse, y sirve como punto de partida para convertir la cotización en una venta real. Facilita el seguimiento comercial y agiliza el proceso de atención al cliente.

### Clientes y Ventas (Historial)

En este módulo se administra la información de los clientes y se visualiza el historial de ventas realizadas. Permite consultar datos de contacto, revisar compras previas y verificar el estado de cada operación. Es útil para mantener un registro ordenado de clientes frecuentes y para dar seguimiento a las ventas ya procesadas.

### Comprobantes

El módulo de Comprobantes permite generar, registrar y consultar comprobantes electrónicos asociados a las ventas, como facturas o boletas. Desde aquí el usuario puede visualizar el documento, validar su información y revisar el estado de emisión. También organiza los comprobantes emitidos para un control administrativo más eficiente.

### Guías de Remisión

Este módulo gestiona la emisión de guías de remisión remitente, necesarias para el traslado de los inflables o productos vendidos. Permite registrar datos como punto de partida, destino, transportista y bienes transportados. Facilita el cumplimiento de requisitos logísticos y tributarios relacionados con el traslado de mercadería.

### Notas de Crédito

Este módulo permite crear notas de crédito cuando es necesario anular o modificar un comprobante ya emitido, sea por error, devolución o ajuste. Se registra el motivo, el comprobante relacionado y los datos necesarios para dejar constancia formal del cambio. Garantiza un manejo correcto y ordenado de los documentos rectificatorios.

### Historial de Notas de Crédito

Aquí se pueden consultar todas las notas de crédito generadas en el sistema. El historial permite revisar fechas, motivos, comprobantes vinculados y detalles de cada documento. Su propósito es mantener un control transparente de los ajustes realizados y facilitar auditorías o revisiones posteriores.
